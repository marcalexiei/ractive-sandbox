# Ractive Sandbox

Project used to test Ractive.

## Typings

1. Open VSCode
2. Run `npm i`
3. Open `main-typings.ts` and check errors

## Class props

1. Open VSCode
2. Run `npm i`
3. Run `npm run build:class-props`
4. Open `ractive-with-class-props/index.html` inside your browser and check the console
